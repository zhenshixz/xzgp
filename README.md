# 📈 A股股票ST/退市风险诊断与基本面分析系统

这是一个基于 **Python Flask + 东方财富 API** 的全栈股票基本面诊断与退市风险防范系统。它特别针对有 ST 风险、违规及退市警示的 A 股股票进行实时筛查与深度诊断。

---

## 🛠️ 技术栈

*   **后端**：`Python` + `Flask` + `curl_cffi`（智能绕过安全防爬校验） + `BeautifulSoup`
*   **数据源**：东方财富网网关接口、东财股吧公告及资讯数据
*   **部署/启动**：支持 `Vercel` 部署（内含 `vercel.json`）及本地局域网一键启动脚本（`run_intranet.bat`）

---

## 🌟 核心功能

*   🔍 **全局股票联想搜索**：实时调取东财搜索建议接口，输入股票名称/拼音/代码即可快速定位。
*   📊 **基本面多维透视**：自动提取上市交易所、核心主营业务、经营范围、行业板块及法人董事长等公司概况。
*   💸 **单季度净利润深度诊断**：
    *   系统通过复杂的财报数据解析，将多季度的累计数据自动拆解，计算出**独立的单季度净利润**。
    *   智能算出**同比增幅（YoY）**，帮助投资者提前发现公司财务恶化的征兆。
*   ⚠️ **智能退市风险评估**：
    *   实时抓取**近 1 年**该股在股吧中的公告与资讯。
    *   使用语义过滤器，针对 `ST`、`*ST`、`退市`、`立案调查`、`违规`、`特别处理` 等敏感词进行高频筛查。
    *   给出 **Critical（极高风险）/ High（高风险）/ Medium（中风险）/ Low（低风险）** 四档风险等级与诊断报告。
*   📈 **概念题材与 60 日 K 线展示**：关联展现股票的题材标签，并动态渲染历史行情走势。
